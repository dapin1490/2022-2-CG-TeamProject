## 협업 방식
- 명명 규칙 : 파스칼케이스(예: `PascalCase`)
- 파일 및 폴더 정리 : `Materials`, `Scripts`, `Shaders` 폴더에 각자 분리. 새 폴더가 필요할 때, 빈 폴더로 두면 깃허브 pull/push에 오류가 생길 수 있으니 `.placeholder` 만들어두기.
- 커밋 메시지 : 어떤 부분을 변경하였는지 또는 무엇을 하였는지 간략하게 적는 것을 권장.
